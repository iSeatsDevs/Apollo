## Currency

If you want to display the input as a currency. You can pass an object to the `currency` prop
with a `locale` and `format` property. The locale is any usable locale format such as `nl-NL`
where the `format` is the currency formatting that will be used such as `EUR`. E.g:
`{ locale: 'nl-NL', currency: 'EUR' }`
