## Default value

If you need to set a default value for the component you can use the 
`defaultValue` prop. For single pick select components you can simply 
set the prop to whatever value one of your `options` has. If you have
set the `multiple` prop, you can send and array of values like:
`['1', '2', '3']`.

<Preview>
  <Story name="Default Value">
