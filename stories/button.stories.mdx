# Button

We use buttons for any business logic that requires user input. Actions include but are not limited to navigation, confirmation, modals.

## Icon

You can pass any node to the `icon` prop to add a inline icon or element to the button. You can push the icon to the
right side by using `iconRight`.

<Preview>
  <Story name="Icon">
    <>
    <Button icon={<CheckCircle/>} onClick={action('Clicked')}>With Icon</Button>
    <Button icon={<CheckCircle/>} iconRight onClick={action('Clicked')}>With Icon</Button>
    </>
  </Story>
</Preview>
